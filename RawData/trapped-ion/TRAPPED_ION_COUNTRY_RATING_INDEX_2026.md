# Trapped-Ion Country Rating Index (TICRI-2026)

Date: 2026-03-07

## Framework

Each country is scored 1-5 on six dimensions, then converted to a weighted 0-100 score.

Dimensions and weights:
- `Policy & Funding` (25%)
- `Research Depth` (20%)
- `Industry Strength` (20%)
- `Deployment Infrastructure` (15%)
- `Global Integration` (10%)
- `Commercial Momentum` (10%)

Formula:
`Score (0-100) = ((P*0.25 + R*0.20 + I*0.20 + D*0.15 + G*0.10 + C*0.10) / 5) * 100`

Rating bands:
- 85-100: Global Leader
- 70-84: Strong Contender
- 55-69: Emerging Power
- 40-54: Developing Ecosystem
- <40: Early Stage
